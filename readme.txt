== Description ==
This plugin adds a jQuery Quote Rotator to a site. Users would see this Quote Rotator before any content during their first visit and they can click to opt out/continue. If the user chooses to opt out, she won't see it again until her session expires. The admin is able to add, edit and delete Quotes. The jQuery Quote Rotator is opensource and can be found at http://tympanus.net/codrops/2013/03/29/quotes-rotator/

== Installation ==
Upload plugin dir to the /wp-content/plugins/ directory
Activate the plugin through the [Plugins] menu in WordPress
