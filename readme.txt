== Description ==
This plugin displays a jQuery Quote Rotator before you are able to view any page on a WordPress blog/site for the first time you visit the site. 
If you choose to opt out of the quote rotator, you won\'t see it again until your session expires. The admin is able to add, edit and delete quotes.
The jQuery Quote Rotator is opensource and it can be found at http://tympanus.net/codrops/2013/03/29/quotes-rotator/

== Installation ==
Upload plugin dir to the `/wp-content/plugins/` directory
Activate the plugin through the \'Plugins\' menu in WordPress
